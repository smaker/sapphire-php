## 설치 환경
- PHP 7.4.0 이상
- MySQL 5.0.7 이상 또는 MariaDB 권장
- 필수 PHP 모듈
  - curl
  - gd
  - mbstring
  - openssl
  - json
  - xml 및 simplexml
  - php.ini에서 session.auto_start = Off로 설정되어 있어야 합니다.
