쾌적하고 사용자 친화적인 CMS

## 설치 환경
- PHP 5.3.3 이상 (PHP 5.5.9 이상 권장, PHP 7 지원)
- MySQL 4.1.13 이상 (MySQL 5.0.7 이상 또는 MariaDB 권장)
- 필수 PHP 모듈
  - curl
  - gd
  - mbstring
  - mcrypt
  - json
  - xml 및 simplexml
  - php.ini에서 session.auto_start = Off로 설정되어 있어야 합니다.
